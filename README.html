Local Development ​

Environment requirements:

  • Node.js 18.0.0+

1. Enable corepack & pnpm ​

If your Node.js version is lower than 16.13.0，Please install corepack first.

shell
npm install -g corepack
shell
corepack enable
corepack prepare pnpm@latest --activate

2. Clone the repository ​

shell
git clone https://github.com/cworld1/ccp-learning.git
cd ccp-learning

2. Install dependencies ​

shell
pnpm install

3. Start the development server ​

shell
pnpm run dev

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

4. Some useful commands ​

pnpm run build Bundles your website into static files for production.
